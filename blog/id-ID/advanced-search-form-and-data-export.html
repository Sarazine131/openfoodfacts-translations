Di bagian atas kanan Open Food Facts sekarang adalah tautan Pencarian Lanjutan. Ini menyediakan akses ke formulir pencarian di mana Anda dapat menentukan banyak kriteria untuk menyaring produk - produk dan memilih hanya yang mengandung atau tidak mengandung merek, label, asal usul, kode pengemas, zat aditif, dll.

Anda juga dapat menentukan nilai batasan untuk setiap unsur komposisi nutrisi: energi, protein, lemak, karbohidrat, dan semua mineral dan vitamin.

Anda juga akan menemukan pada akhir formulir ini sebuah tombol untuk mengunduh data dalam format csv. Anda dapat membaca format ini di Excel, Perangkat Lunak Gratis LibreOffice, Open Refine, dan perangkat lunak lainnya.

Itulah gunanya data terbuka: mereka tersedia dalam format yang memungkinkan pengunaan kembali data dengan mudah. Anda dapat menyortir, menggabungkan, membandingkan, dan sebagainya.

Jika Anda melihat hal - hal menarik dalam data, atau jika Anda menemukan kesalahan, beri tahu kami. :-)

Detail teknis (yang mungkin diminta saat membuka data oleh perangkat lunak Anda, jika ia tidak secara otomatis mengenali formatnya):
- pengodean karakter: UTF-8
- pemisahan field berdasarkan tab (tab)
